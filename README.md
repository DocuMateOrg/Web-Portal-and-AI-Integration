## Project Overview

Documate is an AI-powered platform for document processing, enabling:

- OCR for multilingual text (Sinhala & English)  
- AI-based summarization and tagging  
- Semantic search  
- Audio playback of document summaries  

This system combines advanced AI integration with a user-friendly web portal to streamline document management and accessibility.

---

## Features

### AI Integration (Python + Gemini)

- **Image Preprocessing:** Deskew, denoise, brightness correction, binarization  
- **Gemini Vision OCR:**  
  - Sinhala & English text extraction  
  - Structure detection  
  - Confidence scoring  
- **Gemini Text:**  
  - Summary generation  
  - Tags & categories  
  - Language detection  
  - Batch multi-document processing  
- **Google TTS:** Generate audio summaries (MP3) and upload to Firebase Storage  

### Web Portal (React)

- Login & dashboard  
- Document list and detailed document viewer  
  - OCR text  
  - Summaries  
  - Tags  
  - Audio playback  
- Search UI with filters  
- Group management & share permissions
